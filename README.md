# MindHub AI

MindHub AI is a full-stack AI-powered interview preparation web application.

It helps users create interview sessions, practice questions, and get AI-generated guidance using Gemini AI.

## Features

- User registration and login
- JWT authentication
- AI-powered interview question generation
- MongoDB database connection
- Protected backend routes
- React frontend with Vite
- Express.js backend API
- Clean project structure

## Tech Stack

### Frontend
- React
- Vite
- JavaScript
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Gemini API
